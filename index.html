LinkedIn Auto Post Agent

Full Stack / API Dev
Tech Stack
Node.js Express.js LinkedIn API OAuth 2.0 OpenID Connect REST APIs
Overview

A backend automation agent that programmatically publishes posts to LinkedIn using the official LinkedIn API. The system manages authentication, secure token exchange, and automated publishing through a structured backend workflow.

Core Implementation
  • OAuth Authentication: Implemented complete 3-Legged OAuth flow, authorization code exchange, and handled scopes (openid, profile, w_member_social).
  • User Identity Handling: Retrieved authenticated member ID using the OpenID endpoint.
  • Automated Publishing: Built a backend module to create and publish posts with structured JSON payloads.
  • Security: Stored credentials securely using environment variables (.env).
